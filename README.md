# 🏝️ Treasure Island Adventure Game

An interactive **Treasure Island adventure game** built using **Python and Streamlit**.
Players must make the correct sequence of choices to successfully reach the hidden treasure.

🚀 **Live App:**
https://treasure-island-game-vnwmjhnkxqiksga26y6as4.streamlit.app

---

## 🎮 About the Game

Treasure Island is a simple decision-based adventure game where the player starts on a mysterious island and must choose different paths to reach the treasure.

Each decision leads to a different outcome. Some paths lead to **victory**, while others result in **Game Over**.

The game demonstrates how **Python logic and Streamlit UI components** can be used to create interactive web applications quickly.

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/yourusername/treasure-island-game.git
```

Move to the project directory:

```
cd treasure-island-game
```

---

## ▶️ Run the Application

Install dependencies:

```
pip install -r requirements.txt
```

Run the Streamlit app:

```
streamlit run app.py
```

Open in browser:

```
http://localhost:8501
```

---

## ⚡ Quick Start (One Command)

You can also run everything using a single command:

```
python -m venv venv && venv\Scripts\activate && pip install -r requirements.txt && streamlit run app.py
```
